# SharpUtils

SharpUtils — это набор утилит для .NET, предназначенный для безопасной и функциональной обработки ошибок, композиции результатов и повышения удобства разработки.

## Results

Библиотека — [SharpUtils.Results](Results/README.ru.md) — реализует паттерн Result (монаду Either), позволяя явно и безопасно обрабатывать успех и ошибку операций.

- [Документация Results (RU)](Results/README.ru.md)
- [Документация Results (EN)](Results/README.md)

---

### Быстрый старт

- Поддерживаемые платформы: .NET 9, 8, 7, 6, .NET Standard 2.1
- Установка:

```
dotnet add package SharpUtils.Results
```
