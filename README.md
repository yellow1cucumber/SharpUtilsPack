# SharpUtils

SharpUtils is a set of utilities for .NET, featuring the Results library for functional error handling and result composition.

## Results

The library — [SharpUtils.Results](Results/README.md) — implements the Result pattern (Either monad), enabling explicit and safe handling of operation success and failure.

### Quick Start

- Supported platforms: .NET 9, 8, 7, 6, .NET Standard 2.1
- Installation:

```
dotnet add package SharpUtils.Results
```
